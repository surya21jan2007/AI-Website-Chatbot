# 🌐 AI Website Chatbot

An AI-powered Website Chatbot built using Streamlit, LangChain, FAISS, HuggingFace Embeddings, and GPT4All.

This application allows users to enter a website URL and ask questions about the website content. The chatbot uses Retrieval-Augmented Generation (RAG) to retrieve relevant information and generate accurate answers.

---

## 🚀 Features

* Website Content Extraction
* Semantic Search
* Question Answering
* RAG Architecture
* FAISS Vector Database
* Source References
* User-Friendly Streamlit Interface

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/AI-Website-Chatbot.git
cd AI-Website-Chatbot
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
